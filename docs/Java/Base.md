# Java基础结构和语法
我们从一个`HelloWorld`开始
```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello World!");
    }
}
```
上面代码的作用仅是从控制台打印一行字符串。

Java类的开头为：访问修饰符+类型+名称。上面的代码声明了一个访问修饰符为`public`,类型为`class`，名称为`HelloWorld`的类。

#### 访问修饰符
访问修饰符 Java中的访问修饰符分为四种:
- public 
- default
- protected
- private

#### 类型
- class 
- interface
- enum
